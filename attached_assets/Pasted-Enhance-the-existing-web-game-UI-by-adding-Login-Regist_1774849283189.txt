- DO NOT change layout, structure, or positions

- ALL TEXT MUST USE INDONESIAN LANGUAGE

========================
TOMBOL KANAN ATAS
========================
- Sebelum login:
  - Tampilkan tombol: "Masuk"

- Setelah login:
  - Tampilkan inisial username (huruf pertama)
  - Contoh: "Dandi" → "D"
  - Bentuk lingkaran (rounded)
  - Warna gold dengan efek glow

- Saat diklik:
  - Tampilkan dropdown atau modal menu profil

========================
MENU PROFIL
========================
Tampilkan informasi:
- Nama Pengguna
- Kata Sandi (disamarkan: ******)
- Bank (READ ONLY / tidak bisa diubah)
- Nama Pemilik Rekening (READ ONLY)
- Nomor Rekening (READ ONLY)

ATURAN:
- Hanya kata sandi yang bisa diubah

Fitur:
- Tombol: "Ubah Kata Sandi"
- Input password baru
- Simpan ke localStorage

========================
RIWAYAT TRANSAKSI
========================
Tambahkan menu atau section:

Judul:
"Riwayat Transaksi"

Isi list:
- Jenis:
  (Deposit / Tarik Dana / Masuk Game / Menang / Kalah)
- Nominal:
  contoh: +$10 atau -$5
- Tanggal & waktu
- Status:
  (Berhasil / Pending / Gagal)

WARNA:
- Hijau = positif (menang / deposit)
- Merah = negatif (kalah / withdraw)
- Gold = highlight utama

FITUR TAMBAHAN:
- Filter:
  (Semua / Deposit / Game / Tarik Dana)

DATA:
- Generate data otomatis
- Tambahkan data saat:
  - Klik "Masuk Game"
  - Simulasi menang/kalah
  - Deposit / Tarik Dana

========================
DAFTAR AKUN
========================
Form:
- Nama Pengguna
- Kata Sandi
- Konfirmasi Kata Sandi
- Pilih Bank:
  (BCA, BRI, DANA, BSI, BNI, GOPAY)
- Nama Pemilik Rekening
- Nomor Rekening

VALIDASI:
- Semua field wajib diisi
- Password harus sama dengan konfirmasi

ATURAN:
- Setelah daftar:
  - Data bank & rekening menjadi permanen (tidak bisa diubah)

- Setelah berhasil daftar:
  - User otomatis login
